# OCRByMe - 快速开始指南

## 🚀 一键使用（推荐）

### 最简单的方法：

1. **复制 PDF 文件到项目文件夹**
   ```
   复制你的 PDF → ocrbyme 文件夹
   ```

2. **双击运行批处理文件**
   ```
   双击 run_ocr.bat
   ```

3. **完成！**
   - ✅ 结果保存在 `out` 文件夹
   - ✅ 内容已复制到剪贴板（按 Ctrl+V 粘贴）
   - ✅ 输出文件夹自动打开

---

## 📁 文件说明

- **run_ocr.bat** - 双击这个文件开始转换
- **out/** - 输出文件夹（自动创建）
- **批处理使用说明.md** - 详细使用说明

---

## ⚠️ 注意事项

- 一次处理一个 PDF 文件
- 测试文件（test_document.pdf、enriched_test_document.pdf）会被自动跳过
- 确保 `.env` 文件中已配置 API Key

---

## 🎯 完整工作流程

```
1. 放入 PDF 文件
   ↓
2. 双击 run_ocr.bat
   ↓
3. 等待处理（显示进度）
   ↓
4. 自动完成：
   - 生成 Markdown 文件（out 文件夹）
   - 复制到剪贴板
   - 打开输出文件夹
   ↓
5. 按 Ctrl+V 粘贴内容
```
